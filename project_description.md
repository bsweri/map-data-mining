GeoExtract automates B2B lead generation by extracting contacts using legal Google Maps API,saving businesses countless manual hours. 

Engineered for scalability and performance, this project showcases a modern tech stack. It's built with React, TypeScript, and Vite for a highly optimized, type-safe frontend, styled with Tailwind CSS. The secure backend is powered by Supabase (PostgreSQL, Edge Functions, Auth), utilizing React Router for seamless navigation, and integrating Midtrans & LemonSqueezy APIs to handle complex payment processing workflows.
